### 🔥 Featured Projects

#### 🛠️ [Serviquo – Home & Appliances Service Request Platform](#)
**Tools:** Python, Django, HTML, CSS, JavaScript, SQLite

A centralized, secure web platform that digitizes how users request home and appliance services — replacing unorganized, phone-based booking with a structured, privacy-first workflow.

- Architected a **6-role system** (Admin, User, Service Provider, Company, Technician, Shop) with role-based access control, so each stakeholder gets a purpose-built dashboard
- Designed a **relational database with 20+ interlinked tables** in SQLite to manage users, service requests, technician assignments, leave requests, and shop inventory
- Built end-to-end **service request workflows**: users submit requests and track real-time status, while technicians/companies receive, update, and close them — all without exposing either party's phone number
- Implemented a **Shop module** for browsing and purchasing service-related tools, spare parts, and equipment tied to active service requests
- Enabled **technician leave management** and workload tracking for companies to coordinate assignments efficiently
- Applied **Django's built-in security features** alongside custom validation to protect user data and prevent unauthorized cross-role access
- Conducted **unit, integration, and user acceptance testing** to validate request routing, status updates, and role permissions before deployment

#### 🅿️ [EasyPark – Online Parking Slot Booking System](#)
**Tools:** PHP, HTML, CSS, JavaScript, MySQL

A real-time, web-based parking slot booking system that replaces manual, first-come-first-serve parking management with a transparent, automated digital platform.

- Designed a **normalized MySQL schema** (8 core tables — users, admins, parking plots, slots, bookings, vehicle types, complaints, feedback) with proper primary/foreign key relationships
- Built a **live slot availability system** with visual, color-coded slot selection (booked vs. available) to prevent double-booking in real time
- Developed dual interfaces: a **user portal** for registration, slot browsing, booking, cancellation, and feedback, and an **admin dashboard** with analytics on total users, plots, available slots, and booking trends
- Implemented **booking and revenue reporting** with date-range filters and per-slot pricing, giving admins visibility into utilization and earnings
- Added a **complaints and feedback loop** allowing users to raise issues and admins to respond directly within the platform
- Applied **input validation and session-based authentication** to guard against SQL injection, XSS, and unauthorized access
- Followed the full **SDLC**: requirements specification, DFD-based system design, unit/integration/system/UAT testing, and a phased rollout (pilot run → full deployment)
